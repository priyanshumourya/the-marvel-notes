# 🦸‍♂️ Marvel Notes App

A stylish and interactive Notes App inspired by Marvel design, built using HTML, CSS, and JavaScript. This app allows users to create, edit, delete, and search notes with a smooth UI and animations.

## 🚀 Features
- ➕ Add notes with title and content
- 🔄 Edit existing notes
- 🗑️ Delete notes
- 🔍 Search notes by title
- 📖 Read More / Show Less functionality
- 💾 Data stored in browser using LocalStorage
- 🎨 Marvel-themed UI with animations

## 🛠️ Tech Stack
- HTML5
- CSS3 (Animations & Styling)
- JavaScript (CRUD Operations + LocalStorage)

## 💡 How it Works
- Notes are stored in browser local storage
- CRUD operations are handled using JavaScript
- Dynamic UI updates without page reload
